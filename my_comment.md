Currently, I am building an AI agent called **Cognivise** — a closed-loop cognitive intelligence system designed to *watch, listen, and adapt* to learners in real time. The objective is not passive monitoring, but **adaptive intelligence** that evolves with each learner’s behavior.

🔗 **Project Repository (Cognivise):**

The system is built using an open-source Vision SDK that enables real-time visual understanding:

🔗 **Vision SDK (Vision Agents):**

I strongly believe this NVIDIA session will directly help me strengthen the **real-time visual intelligence and adaptive feedback loop** of Cognivise. My long-term goal is to build an advanced monitoring system that adapts uniquely to each learner — not a one-size-fits-all solution.
